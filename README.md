# Student Grade Tracker

## Description

The **Student Grade Tracker** is a Java console-based application developed as part of the **CodeAlpha Java Programming Internship**. It helps manage student records, calculate grades, determine pass/fail status, and generate academic reports using Object-Oriented Programming (OOP) concepts.

---

## Features

* Add Student Records
* Display All Students
* Search Student by Roll Number
* Edit Student Details
* Delete Student Record
* Manage Department and Section
* Calculate Average Marks
* Find Highest and Lowest Marks
* Assign Grades (A, B, C, D, F)
* Display Pass/Fail Status
* Generate Student Result Summary

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

---
